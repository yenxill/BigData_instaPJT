## 1. 프로젝트 소개

  - 프로젝트 이름 : Instagram
  - 프로젝트 내용 : 실제 Instagram SNS 디자인과 기능을 구현

### 개발 기간
> 2024.10.14 ~ 2024.11.20


## 2. 주요기능


  - 로그인/로그아웃/회원가입
  - 피드 등록/수정/삭제
  - 피드 좋아요/댓글
  - 사용자의 프로필 피드 확인


## 3. 화면 및 주요 기능
